# RetroLog

Your complete archive of taste — track films, games, books, albums, and artists in one beautiful social platform.

## Stack
- Next.js 14 + TypeScript
- Tailwind CSS
- Supabase (Auth + DB + Storage)
- Vercel (Deployment)

## Getting Started

```bash
npm install
npm run dev
```

Copy `.env.local.example` to `.env.local` and fill in your Supabase credentials.
