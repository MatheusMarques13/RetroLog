# RetroLog

Your complete archive of taste — track films, games, books, albums, and artists in one beautiful social platform.

## What is RetroLog?

RetroLog is a Letterboxd-inspired platform for tracking every kind of media consumption across five categories:

- Films
- Games
- Books
- Music Albums
- Artists

## Current State

This repo currently contains the landing page (`index.html`) — a single-file prototype with:

- Hero section with animated floating icons and scanline effects
- Media tabs showcasing Films, Games, Books, Albums, and Artists
- Featured review card
- Stats marquee bar
- How it works section
- Recent activity feed
- Sign up and browse pages
